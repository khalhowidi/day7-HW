# flutter_application_1

Store App
Project Description
 Store is a Flutter application built as a final project.
The app allows users to create an account, log in, browse products, view product details, and add items to a shopping cart.

The application uses Supabase for authentication and database management.

---

Screens
The application includes at least 5 screens:

Sign Up Screen
Log In Screen
Products Screen
Product Details Screen
Cart Screen

---

Features
User authentication using Supabase (Sign Up & Log In)
Retrieve product data from Supabase database
Display products using models and JSON parsing
Navigation between screens
Add and remove items from cart
Loading indicator while fetching data
Error handling for data loading

---

Technologies Used
Flutter
Dart
Supabase
Supabase Flutter package

---

Database
The project uses Supabase to:

Authenticate users
Store product data in a database table
Retrieve data from the database
